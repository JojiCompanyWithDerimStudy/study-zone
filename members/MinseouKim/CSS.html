선택자 연습 홈페이지

Font-size속성

    PX과 REM의 차이

  • REM : html의 폰트 설정의 영향을 받는다. 그래서 글자를 유동적으로 변경하기 좋아 대중적이다.
  • PX : REM 과 달리 고정된 값이기 때문에 폰트의 설정을 받지 않아 많이 사용하지 않는다.

Color 속성

    Color의 색상 지정 방법

  1. 영어
  2. rgb(255,255,255)
  3. 16진수(ex:#001122F)

text-align 속성

    정렬 위치를 설정해주는 속성

  1. right,left : 오른쪽, 왼쪽으로 정렬
  2. center : 가운데 정렬
  3. justify : 텍스트 창의 왼쪽과 오른쪽 간격이 공평하게 정렬됨

font 속성

    폰트와 관련된 여러 속성들을 표현

  1. font-weight :폰트의 두께 설정
  2. line-height : 행과 행 사이 간격을 지정
  3. font-fanmily : 서체를 지정하는 속성

Flex 속성

    레이아웃의 크기를 키우거나 줄이는 방법을 설정하는 속성

  1. display : flex;
  2. flex-direction : row/row-reverse/column/column-reverse/

    id와 class의 소개와 차이

    id와 class를 사용하는 이유는, 특정 요소를 이후에 CSS 또는 Javascript 내에서 선택하기 쉽게 하기 위함입니다.

    id는 특정 요소를 선택하기 위해 사용하는 속성입니다. 하나의 요소에 하나의 id만 가질 수 있습니다.

    class는 특정 요소를 선택하기 위해 사용하는 속성입니다. 하나의 요소에 여러 개의 class를 가질 수 있습니다.

      • global.css를 작성하여 민서님의 문서들에 적용하기.

      • id와 class 차이점 이해하고 이해한점 문서에 정리하기.
